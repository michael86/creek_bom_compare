# creek_bom_converter

This is a simple Electron boilerplate that uses React and Typescript along with fs-extra and lodash bundled in for ease. All of the configs have had their paths set up for electron compiling.

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
